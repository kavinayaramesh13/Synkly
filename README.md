**Synkly**
Synkly is a peer-to-peer skill exchange platform where users can teach skills they know and learn skills they want from others through mutual knowledge exchange.

**Features**
1.User registration and login
2.Secure authentication using JWT
3.Password encryption with bcrypt
4.Skill-based user profiles
5.Peer skill matching system
6.Protected backend APIs
7.PostgreSQL database integration
8.Real-time communication features (planned)
9.AI-powered smart matching (planned)

**Tech Stack**
**Frontend**
1.React
2.Vite
**Backend**
1.Node.js
2.Express.js
**Database**
1.PostgreSQL
**Authentication**
1.JWT
2.bcryptjs

**Project Goal**
The goal of Synkly is to create a collaborative learning ecosystem where people can exchange knowledge directly with each other instead of relying only on paid learning platforms.

Example:
User A teaches Java and wants to learn React
User B teaches React and wants to learn Java
Synkly matches them for a skill exchange session
